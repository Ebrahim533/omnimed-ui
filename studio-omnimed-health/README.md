# OmniMed Sanity Studio

Sanity Studio configuration for managing content for the OmniMed health platform.

## Setup

```bash
cd studio-omnimed-health
npm install
npm run dev
```

The studio will be available at `http://localhost:3333`

## Schemas

- **Person** - Team members and leadership profiles
- **Site Settings** - Global company information and images
- **Social Link** - Social media profile links
